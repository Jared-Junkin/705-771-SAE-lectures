# 705.771 SAE Lectures

This covers the two modules in the Johns Hopkins University course 705.771: introduction to mechanistic interpretability. These correspond to the Anthropic whitepapers "Towards Monosemanticity" and "Scaling Monosemanticity".

## Setup

First, create a virtual environment with `uv venv`. Then, install the required packages by running `uv pip install -r requirements.txt`. Finally, clone the dictionary learning library with `git clone https://github.com/saprmarks/dictionary_learning`.
